# 🛒 Retail Store Sales Performance Analysis

## 📌 Overview
This project analyzes retail sales data using SQL to extract insights about revenue, product performance, and trends.

## 🎯 Objectives
- Identify top-selling products
- Analyze revenue trends
- Understand seasonal demand patterns

## 🛠️ Tech Stack
- MySQL
- SQL Queries

## 📊 Key Features
- Complex SQL queries (JOIN, GROUP BY, ORDER BY)
- Revenue analysis
- Product performance tracking

## 📈 Insights
- Top-performing products identified
- Monthly and seasonal sales trends analyzed
- Revenue optimization insights generated
